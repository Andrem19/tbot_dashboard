# 06. Стили и UI

## Система стилей

Основные стили находятся в `src/index.css`.

Используются CSS custom properties:
- фоны: `--bg-main`, `--bg-panel`, `--bg-input`;
- текст: `--text-primary`, `--text-secondary`;
- акценты: `--accent`, `--green`, `--red`;
- шрифты: `--font-sans`, `--font-mono`.

`src/App.css` содержит базовые/наследованные стили от шаблона Vite, которые почти не влияют на итоговый интерфейс.

## Layout

Вертикальные секции экрана:
1. `form-row` (шапка фильтров + PnL)
2. `main-content` (график/табы)
3. `report-strip`
4. `heatmap-container`
5. `status-bar`

Ключевой приём для корректного поведения графика: использование `min-height: 0` в flex-контейнерах.

## Адаптивность

- `< 768px`:
  - шапка перестраивается в grid;
  - heatmap показывает недельный вид;
  - увеличены размеры интерактивных элементов.
- `>= 769px`:
  - heatmap переключается на месячный вид (31 колонка);
  - weekly-вид скрывается.

## Визуальные индикации

- PnL: зелёный/красный по знаку.
- Status bar:
  - WS connected/disconnected;
  - DB ON/OFF;
  - кнопка `Reconnect WS`, если WS не подключён.
- График:
  - `Entry` (синий), `SL` (красный), `TP` (зелёный).
