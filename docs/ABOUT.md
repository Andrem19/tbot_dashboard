# О проекте

`tbot_dashboard` — это UI-дашборд для визуализации торговли бота на торговых свечах.

## Суть проекта

- Бот торгует отдельно и сам пишет данные в Firebase Realtime Database.
- Этот проект ничего не исполняет на стороне торговли.
- UI только читает данные из Firebase, подгружает рыночные свечи Binance и визуализирует состояние в браузере.

## Что показывает UI

- текущую активную позицию бота (entry/sl/tp, side, qty, текущий PnL);
- свечной график инструмента;
- маркеры истории сделок на графике;
- агрегированные метрики отчёта (`report`);
- историю сделок по дням в heatmap.

## Источники данных

1. Firebase Realtime Database (путь `dashb`) — данные бота.
2. Binance Futures REST/WebSocket — рыночные свечи для графика.

## Границы ответственности

Проект:
- читает данные;
- отображает данные;
- даёт визуальный контроль состояния.

Проект не:
- открывает и не закрывает сделки;
- не отправляет торговые команды на биржу;
- не изменяет данные бота в Firebase.
